Project Idea : Bet’it

Problem: A lot of betters are using social networks (Twitter, Snapchat, Facebook, Instagram) to share their bets that may be lost in users’ feed.

Idea: Build an app to dedicated to bets.

2 types of users :

  • Classical user that will look for bet and maybe pay a subscription
  • Better which is classical user. The classical user can be a better by building a request that will be validated by the Bet’it staff. He can publish bets and be paid.

User can

  • Follow a better and see bets of this better
  • See available bets sorted by date through a quick view / feed
  • Follow a bet, he’ll be able to see through a specific section, his bets in ‘progress’
  • See success rate of a better
  • Do search by sport / tags

Better can

  • Publish bets
  • Put on his profile his favorite sports and a small description
  • same as user

MVP

  • User registration / User login
  • Request system to move from user to better
  • Ability for betters to publish bets
  • Ability for users to follow betters
  • Feed section to see available bets
  • Profile page (betters only)

Others features

  • Follow a bet to have it in the ‘in progress’ feed
  • Feed section to see bets in progress
  • See success rate / analytics of a better
  • Form to submit the addition of a new sport
  • Back office / manage part
  • Website
  • Payment system
